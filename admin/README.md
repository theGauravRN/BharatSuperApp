# BharatSuperApp — Admin Panel

Internal admin dashboard for managing BharatSuperApp.

## Stack (Planned)
- **Framework**: Next.js 14 (App Router)
- **Auth**: Supabase + Role-based access (super-admin, moderator, support)
- **UI**: Tailwind CSS + shadcn/ui
- **Charts**: Recharts / Tremor

## Planned Modules
- User management & KYC review
- Content moderation (posts, videos, news)
- FinTech transaction monitoring
- Grievance management dashboard
- Module analytics & metrics
- Role & permission management

## Setup
```bash
cd admin
npm install
npm run dev
```
